

## 🔧 Environment Setup Requirements

Before running this project locally, make sure the following dependency is properly installed:

---

## 1️⃣ NVAPI

The NVAPI package is included inside:

```
Environment Setup/NVAPI.rar
```

### 📦 Installation Steps:
1. Move `NVAPI.rar` **outside** the project directory.
2. Extract the `.rar` file.
3. After extraction, a folder named `NVAPI` will appear.
4. Move the entire `NVAPI` folder directly to your `C:` drive.
   Example:

   ```
   C:\NVAPI
   ```
### ⚠️ Important

* Do **not** place the `NVAPI` folder inside another directory.
* It must be located directly under the root of the `C:` drive.
* Incorrect placement may cause build or runtime errors.
